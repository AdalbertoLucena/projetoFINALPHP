# Projeto CRUD - Restaurante

Arquivos ajustados e prontos para rodar no XAMPP.

Como usar:
1. Copie a pasta projetocrud2 para C:\\xampp\\htdocs\\
2. Inicie Apache e MySQL
3. Importe banco_restaurante.sql no phpMyAdmin
4. Acesse http://localhost/projetocrud2/admin/pratos.php
